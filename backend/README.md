# backend

使用：1、pip install -r requirements.txt 安装依赖（国内镜像：pip install --default-timeout=6000 -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt）

2、python manage.py runserver 启动服务，内置数据库sqlite已初始化，可以直接使用
